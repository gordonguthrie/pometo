Description

Compiles pometo into an Erlang file. Pometo is both an interpreted and compiled language. This module compiles Pometo source files first to Erlang source files and then to .beam files.

Function Index

Function Details

compile/3

compile(Functions, ModuleName, Str) -> any()

The compiler function. This is a partial implementation and is only currently used in the test suite to build an Erlang source file, compile it on the fly and build and load the .beam file. In the fullness of time it will build and persist a .beam file an optionally dump out the Erlang source
