Description

This is the interface of the pometo language to external tools. Pometo runs in two modes:

  • interpreted
  • compiled

    • This module will therefore finally expose two functions:

  • interpret
  • compile

    • (the compile function is currently not implemented)

    A series of other functions are exposed for TESTING ONLY and marked with the postfix _TEST.
